# Benzak Bot

![Benzak Bot](assets/img.png)

**Benzak Bot** следит за доступностью бензина рядом с нужной точкой на карте.

Укажи координаты и радиус поиска, а бот будет присылать оповещения, когда поблизости появляется доступное топливо. Он задуман как тихий помощник для ситуаций, где важно быстро узнать, что бензин снова есть не где-то вообще, а именно рядом с тобой.

## Что делает бот

- принимает координаты точки поиска;
- учитывает заданный радиус;
- отслеживает появление доступного бензина поблизости;
- отправляет уведомления в Telegram, когда есть подходящее совпадение.

## Идея

Меньше ручной проверки, меньше лишних поездок, больше шансов поймать доступную заправку вовремя.
